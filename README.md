# 🌟 سیستم مدیریت محتویات Django - پوریا باغبان

یک سیستم مدیریت محتویات (CMS) تمام عیار و حرفه‌ای بر اساس Django برای مدیریت وب‌سایت پوریا باغبان.

## ✨ ویژگی‌های اصلی

### 📝 سیستم مدیریت بلاگ
- **نویسندگان مختلف**: پشتیبانی برای چندین نویسنده با مجوزهای مختلف
- **وضعیت پست‌ها**: پیش‌نویس، منتشر شده، آرشیو
- **سیستم دسته‌بندی**: سازماندهی پست‌ها بر اساس دسته‌بندی
- **برچسب‌ها**: تگ‌های مختلف برای هر پست
- **نظرات**: سیستم نظرات با تایید خودکار
- **آمار**: نمایش تعداد بازدید و پسندیدگی
- **تصاویر**: پشتیبانی برای تصاویر شاخص

### 📊 داشبورد آماری
- **نمایش ریئل‌تایم آمار**: بروزرسانی لحظه‌ای تمام اعدادها
- **آمار کلی**: تعداد پست‌ها، کامنت‌ها، پیام‌ها، نویسندگان
- **آمار زمانی**: شمارش فعالیت‌های امروز و این هفته
- **لیست‌های اخیر**: نمایش پست‌های جدید، کامنت‌های منتظر، پیام‌های خوانده‌نشده
- **محبوب‌ترین**: نمایش محبوب‌ترین پست‌ها و نویسندگان برتر
- **رابط زیبا**: طراحی مدرن و ریسپانسیو

### 📧 سیستم مدیریت پیام‌های تماس
- **دریافت پیام‌ها**: ذخیره تمام پیام‌های تماس
- **سیستم پاسخ**: پاسخ دادن به پیام‌ها از درون پنل مدیریت
- **نشانه‌گذاری**: مشخص کردن وضعیت خوانده‌شدن و پاسخ‌داده‌شدن
- **جستجو**: جستجو در پیام‌ها بر اساس نام، ایمیل، موضوع

### 🎨 سیستم CMS (30 بخش)
- **صفحات متحرک**: تغییر محتوای صفحات بدون کد
- **30 بخش مختلف**: بخش‌های مختلف برای صفحه اصلی و دیگر صفحات
- **مهارت‌ها**: نمایش مهارت‌ها با درصد
- **لینک‌های اجتماعی**: مدیریت لینک‌های شبکه‌های اجتماعی
- **تنظیمات سایت**: تغییر معلومات کلی سایت

### 🔐 سیستم احراز هویت و دسترسی
- **کاربران مختلف**: پشتیبانی برای مدیران و نویسندگان
- **گروه‌های کاربری**: 
  - **Blog Authors**: می‌تواند پست بنویسد اما admin منتشر می‌کند
  - **Blog Managers**: کنترل کامل بر روی بلاگ
- **مجوزهای سفارشی**: مجوزهای دقیق‌تر برای هر کار
- **محدودیت دسترسی**: دسترسی فقط برای کاربران مجاز

## 🚀 نحوه شروع

### الزامات
- Python 3.8+
- Django 4.0.10
- SQLite3

### نصب و راه‌اندازی

1. **کلون کردن پروژه**:
```bash
git clone <repository-url>
cd pouriabaghban3
```

2. **ایجاد محیط مجازی**:
```bash
python -m venv env
env\Scripts\activate  # Windows
source env/bin/activate  # macOS/Linux
```

3. **نصب وابستگی‌ها**:
```bash
pip install -r requirements.txt
```

4. **مهاجرت پایگاه‌داده**:
```bash
python manage.py makemigrations
python manage.py migrate
```

5. **ایجاد کاربر مدیر**:
```bash
python manage.py createsuperuser
```

6. **ایجاد گروه‌های کاربری**:
```bash
python setup_permissions.py
```

7. **ایجاد داده‌های تست** (اختیاری):
```bash
python create_test_data.py
```

8. **راه‌اندازی سرور**:
```bash
python manage.py runserver
```

9. **دسترسی به پنل**:
- پنل مدیریت: http://localhost:8000/admin/
- صفحه اصلی: http://localhost:8000/
- داشبورد: http://localhost:8000/dashboard/
