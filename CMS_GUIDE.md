## 🔐 سیستم نویسندگان بلاگ

### **مفهوم**
- ✅ نویسندگان می‌توانند **پست‌های بلاگ** بسازند
- ❌ فقط **Admin** می‌تواند **منتشر یا آرشیو** کند
- ✅ نویسندگان صرفاً می‌توانند **پیش‌نویس** (Draft) بسازند

### **مدل‌های بلاگ**

#### 1. **BlogAuthor** - نویسنده بلاگ
```python
- user (OneToOne) → کاربر Django
- bio (TextField) → بیوگرافی
- profile_image (ImageField) → عکس پروفایل
- social_links (JSONField) → لینک‌های اجتماعی
- is_active (Boolean) → فعال/غیرفعال
- post_count() → تعداد پست‌های منتشر شده
```

#### 2. **BlogCategory** - دسته‌بندی
```python
- name → نام دسته‌بندی
- slug → نشانی
- description → توضیح
- color → رنگ برچسب
```

#### 3. **BlogTag** - برچسب
```python
- name → نام برچسب
- slug → نشانی
```

#### 4. **BlogPost** - پست
```python
- title → عنوان
- slug → نشانی
- author → نویسنده (FK)
- category → دسته‌بندی (FK)
- content → متن پست
- excerpt → خلاصه
- image → تصویر
- tags → برچسب‌ها (M2M)
- status → وضعیت (Draft/Published/Archived)
- is_featured → برجسته
- published_at → تاریخ انتشار
- views → تعداد بازدیدها
- likes → تعداد پسندیدگی
```

#### 5. **BlogComment** - نظرات
```python
- post → پست (FK)
- author → نویسنده
- email → ایمیل
- content → متن نظر
- is_approved → تایید شده
```

## 👥 سیستم مجوزها (Permissions)

### **گروه Blog Authors (نویسندگان)**
```python
✅ add_blogpost     - ایجاد پست جدید
✅ change_blogpost  - ویرایش پست
✅ delete_blogpost  - حذف پست
❌ can_publish      - منتشر کردن (فقط مدیر)
❌ can_archive      - آرشیو کردن (فقط مدیر)
```

### **گروه Blog Managers (مدیران)**
```python
✅ تمام مجوزها
✅ can_publish      - منتشر کردن
✅ can_archive      - آرشیو کردن
```

## 🎨 Admin Panel Features

### **BlogPostAdmin**
- **لیست نمایش**: عنوان، نویسنده، دسته‌بندی، وضعیت، بازدیدها
- **Actions**:
  - 📤 منتشر کردن انتخاب‌شده‌ها
  - 📝 برگرداندن به پیش‌نویس
  - 📦 آرشیو کردن
  - ⭐ برجسته کردن
- **فیلتر**: وضعیت، دسته‌بندی، تاریخ، نویسنده
- **جستجو**: عنوان، محتوا، خلاصه

### **BlogAuthorAdmin**
- **لیست نمایش**: نام نویسنده، ایمیل، تعداد پست‌ها
- **فیلتر**: فعال/غیرفعال، تاریخ ایجاد
- **جستجو**: نام، نام‌کاربری، ایمیل

### **BlogCategoryAdmin**
- رنگ‌بندی برای شناسایی بهتر
- تعداد پست‌های منتشر شده

### **BlogCommentAdmin**
- تصویب/رد نظرات
- فیلتر بر اساس تایید‌شده‌ها

## 📝 چگونه استفاده کنیم

### **مرحله 1: ایجاد نویسنده جدید**
1. Admin → Users
2. Create User (مثلا: "علی")
3. Set password
4. Save

### **مرحله 2: اضافه کردن به گروه**
1. Admin → Users → Select User
2. Groups → Add "Blog Authors"
3. Save

### **مرحله 3: نویسنده می‌تواند**
1. Login با اکاونت خود
2. Blog → Add BlogPost
3. شامل: عنوان، دسته‌بندی، محتوا، تصویر
4. وضعیت = Draft (پیش‌نویس)
5. Save

### **مرحله 4: Admin منتشر می‌کند**
1. Admin → Blog → BlogPosts
2. انتخاب پست‌های Draft
3. Actions → Publish
4. Status = Published
