# 📝 راهنمای مدیریت بلاگ

## نمای کلی

این راهنما نحوه استفاده از سیستم مدیریت بلاگ را توضیح می‌دهد.

## 🔑 ویژگی‌های بلاگ

### 1. پست‌های بلاگ
- **عنوان و نشانی**: هر پست یک عنوان و یک slug منحصر به فرد دارد
- **محتوا**: متن اصلی پست
- **خلاصه**: خلاصه‌ای کوتاه از پست برای صفحات لیست
- **تصویر شاخص**: تصویر برای نمایش در لیست‌ها
- **نویسنده**: نویسنده‌ای که پست را نوشته است
- **دسته‌بندی**: دسته‌بندی پست
- **برچسب‌ها**: تگ‌های متعدد برای هر پست
- **وضعیت**:
  - 📝 **پیش‌نویس**: پست هنوز منتشر نشده است
  - ✅ **منتشر شده**: پست برای تمام بازدیدکنندگان قابل مشاهده است
  - 📦 **آرشیو شده**: پست پنهان شده است
- **برجسته**: نمایش پست در بخش برجسته
- **بازدیدها**: تعداد بارهایی که پست دیده شده
- **پسندیدگی**: تعداد بارهایی که پست پسند شده

### 2. نویسندگان
- **کاربر مرتبط**: هر نویسنده با یک کاربر Django مرتبط است
- **بیوگرافی**: معلومات کوتاه درباره نویسنده
- **تصویر پروفایل**: عکس نویسنده
- **لینک‌های اجتماعی**: پیوندهای شبکه‌های اجتماعی
- **وضعیت فعال**: آیا نویسنده فعال است یا خیر

### 3. دسته‌بندی‌ها
- **نام**: نام دسته‌بندی
- **نشانی**: URL-friendly version
- **توضیح**: توضیح کوتاه
- **رنگ**: رنگ نمایش برای دسته‌بندی

### 4. برچسب‌ها
- **نام**: نام برچسب
- **نشانی**: URL-friendly version

### 5. نظرات
- **نویسنده**: نام نویسنده نظر
- **ایمیل**: ایمیل نویسنده
- **متن**: محتوای نظر
- **وضعیت تایید**: آیا نظر تایید شده است یا خیر

## 🔐 سیستم مجوزها

### گروه Blog Authors
- می‌تواند پست جدید بنویسد
- می‌تواند پست‌های خود را ویرایش کند
- نمی‌تواند پست را منتشر کند (فقط admin می‌تواند)
- نمی‌تواند پست دیگران را ویرایش کند

### گروه Blog Managers
- کنترل کامل بر تمام پست‌ها
- می‌تواند منتشر، پیش‌نویس و آرشیو کند
- می‌تواند برجسته کند
- می‌تواند نظرات را تایید کند

### مجوزهای سفارشی
- `can_publish`: احق انتشار پست
- `can_archive`: احق آرشیو کردن پست

## 📱 استفاده از پنل مدیریت

### نوشتن پست جدید

1. **رفتن به بلاگ**:
   - در پنل مدیریت، به "blog > پست‌های بلاگ" برید
   - دکمه "Add Post" را کلیک کنید

2. **پر کردن اطلاعات**:
   - **عنوان**: عنوان پست (الزامی)
   - **نویسنده**: خود شما خودکار انتخاب می‌شوید
   - **دسته‌بندی**: انتخاب دسته‌بندی (اختیاری)
   - **خلاصه**: خلاصه کوتاه (الزامی)
   - **محتوا**: متن اصلی (الزامی)
   - **تصویر**: انتخاب تصویر شاخص (اختیاری)
   - **برچسب‌ها**: انتخاب یا ایجاد برچسب‌ها

3. **انتخاب وضعیت**:
   - **پیش‌نویس**: برای ذخیره بدون انتشار
   - **منتشر شده**: برای نشان دادن برای تمام
   - **آرشیو شده**: برای پنهان کردن

4. **ذخیره**:
   - دکمه "Save" را کلیک کنید

### تایید نظرات

1. **رفتن به نظرات**:
   - در پنل مدیریت، به "blog > نظرات بلاگ" برید

2. **مشاهده نظرات**:
   - نظرات منتظر تایید نشان داده می‌شوند
   - هر نظر یک وضعیت دارد: ✅ تایید یا ❌ منتظر

3. **تایید یا رد کردن**:
   - چندین نظر را انتخاب کنید
   - از actions در پایین لیست استفاده کنید
   - انتخاب "تایید نظرات" یا "رد کردن نظرات"
   - دکمه "Go" را کلیک کنید

## 📊 آمار بلاگ

### مشاهده آمار
1. به داشبورد بروید
2. بخش "📝 پست‌های بلاگ" را ببینید
3. آمار کلی دیده می‌شود

### محبوب‌ترین پست‌ها
1. به داشبورد بروید
2. بخش "⭐ محبوب‌ترین پست‌ها" را ببینید
3. پست‌ها بر اساس تعداد بازدید مرتب شده‌اند

## 🎯 بهترین روش‌ها

### برای نویسندگان
1. **عنوان مناسب**: عنوان باید جالب و توصیف‌کننده باشد
2. **خلاصه خوب**: خلاصه باید برای جذب مخاطب مناسب باشد
3. **محتوای کیفی**: محتوا باید دقیق و مفید باشد
4. **تصویر مناسب**: تصویر باید با محتوا مرتبط باشد
5. **برچسب‌های مناسب**: استفاده از برچسب‌های مناسب برای تشکیل جوانب
6. **آماده برای نقد**: منتظر بمانید تا admin پست را تایید و منتشر کند

### برای مدیران
1. **بررسی کیفیت**: بررسی محتوای پست قبل از انتشار
2. **قوانین دارند**: مطمئن شوید پست‌ها تحت قوانین سایت هستند
3. **زمان‌بندی**: انتشار پست‌ها در زمان‌های مناسب
4. **نظارت بر نظرات**: حذف نظرات نامناسب
5. **برچسب‌گذاری**: مطمئن شوید که برچسب‌ها صحیح هستند

## ⚠️ نکات مهم

- **اتوسیو (SEO)**: عنوان و خلاصه برای جستجو بهینه‌سازی شده‌اند
- **بازخورد**: نظرات کاربران به بهبود محتوا کمک می‌کند
- **آرشیو**: پست‌های قدیمی می‌تواند آرشیو شوند نه حذف
- **نسخه‌های پیشین**: Django خودکار تاریخ تغییرات را ذخیره می‌کند

## 🐛 رفع مشکلات

### سؤال: چرا پست من منتشر نمی‌شود؟
**پاسخ**: ممکن است:
- وضعیت آن "پیش‌نویس" باشد
- admin هنوز آن را تایید نکرده باشد
- نویسنده غیرفعال باشد

### سؤال: چگونه پست را ویرایش کنم؟
**پاسخ**: 
1. به لیست پست‌ها برید
2. روی عنوان پست کلیک کنید
3. تغییرات را انجام دهید
4. دکمه "Save" را کلیک کنید

### سؤال: آیا می‌تونم پست حذف کنم؟
**پاسخ**: 
1. روی پست در لیست کلیک کنید
2. پایین فرم، دکمه "Delete" را کلیک کنید
3. تایید حذف را بپذیرید
